## 📌 프로젝트 개요

구직자들이 실제 면접 환경을 시뮬레이션하고 면접 역량을 개선할 수 있도록 돕는 웹 서비스입니다.
AI 기술을 활용한 개인화된 면접 경험과 응답 분석 피드백을 제공하는 것을 목표로 설계했으며,
직관적인 UI/UX로 면접 준비의 진입 장벽을 낮추는 데 집중했습니다.

---

## ✨ 주요 기능

| 기능 | 설명 |
|------|------|
| 🤖 **AI 면접관 시뮬레이션** | 실제 면접과 유사한 질의응답 환경 제공 |
| 🎥 **실시간 영상 분석·피드백** | 응답 영상 기반 피드백 |
| 📈 **개인화 성장 리포트** | 사용자별 면접 결과·개선점 리포트 |
| 🗂 **직무별 질문 DB** | 다양한 산업·직무에 맞는 질문 제공 |
| 🖥 **사용자 대시보드** | 면접 이력과 진행 상황을 한눈에 |
